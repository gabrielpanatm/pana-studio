Structură de bază pentru un website Zola

Template-ul include elemente reale pentru editare vizuală: navigație, hero, carduri, pași și footer.

Template-uri clare

Layout-ul de bază stă în base.html, iar homepage-ul extinde acest layout prin Tera.

Framework în temă

Framework-ul global este compilat din themes/pana-studio/sass/css-framework/framework.scss.

CSS per pagină

Stilurile homepage-ului sunt în sass/pagini/index.scss, separat de framework.

Workflow

Un punct de plecare editabil, nu o temă rigidă.

  2. 2.Pană Studio inițializează proiectul.
  3. 3.Editezi vizual template-ul și SCSS-ul.
